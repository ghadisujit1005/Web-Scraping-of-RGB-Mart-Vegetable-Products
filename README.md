RGB Mart Vegetable Web Scraping Project
Project Overview

This project focuses on extracting vegetable product data from the RGB Mart website using Python web scraping techniques. The scraped data is cleaned, analyzed, and visualized to understand product availability and price distribution.

Objectives

Extract product details such as Product ID, Name, Quantity, Price, and Product Link

Store the extracted data in a structured format (CSV / DataFrame)

Perform basic data analysis and visualization

Technologies Used

Python

BeautifulSoup – HTML parsing

Requests – HTTP requests

Pandas – Data cleaning & analysis

Matplotlib & Seaborn – Data visualization
